1、功能说明

    此例程用于演示BLE开发板N32WB45XL_EVB的功能。
    在开发板上有温湿度传感器、G-sensor、触控按键、LCD等设备
    

2、使用环境

    软件开发环境：
        IDE工具：KEIL MDK-ARM 5.26.2.0
    
    硬件环境：
        开发板 N32WB45XL_EVB


3、使用说明
    
    1、时钟源：HSE+PLL
    2、主时钟：144MHz
    3、USART1配置：
            TX  -->  PA9            50MHz，AF_PP
            波特率：115200
            数据位：8bit
            停止位：1bit
            无校验

    4、测试步骤与现象
        a，编译下载代码到开发板，复位运行
        b，打开手机APP（NS_BlueTooth）搜索蓝牙广播（WB452_B38）
        c，连接此设备可以进行回环测试
        d，在LCD上可以看到传感器的输出信息
        e，从串口看打印信息，可以查看蓝牙的状态

4、注意事项
    无
